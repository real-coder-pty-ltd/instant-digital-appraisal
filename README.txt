Creates suburb reports from the Domain API.

== Description ==

You'll need:

* A Google Maps API Key with a billable account set up
* Domain API subscription

== Installation ==

1. Upload the plugin or install it with `composer require real-coder-pty-ltd/instant-digital-appraisal` and activate.

== Frequently Asked Questions ==

= Is this free? =

Yes, but you need a Domain API subscription and a Google Maps API key.
